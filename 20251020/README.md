# 图书销售网站

这是一个基于Flask和SQLite的图书销售网站，具有完整的用户认证、购物车、订单处理、愿望清单等功能。

## 功能特性

1. 用户可以用作者、书名或 ISBN 搜索书籍
2. 用户可以查看书籍的具体信息（页数、出版日期和内容简介）
3. 用户可以把书籍放进"购物车"，在结束购物时可以购买其中的书
4. 在完成订单前，用户可以从购物车中删除书籍
5. 购买书时，用户需要输入账单地址、送货地址和信用卡信息
6. 用户可以对书籍进行评分和发表评论
7. 用户可以创建账户用于保存送货和账单信息
8. 用户可以编辑账户信息（信用卡、送货地址和账单地址等）
9. 用户可以把书籍放入"愿望清单"，网站的其他访问者也可以看到该清单
10. 用户可以把愿望清单中的项目放入自己的购物车
11. 老顾客能够在90秒内找到书籍和下订单
12. 用户可以查看自己的历史订单
13. 查看历史订单时，用户可以方便地重新购买订单中的东西
14. 网站总是告诉购物者她最近三次查看的项目，并提供返回的链接
15. 用户可以看到我们针对各种主题推荐的书籍
16. 用户能够很容易查找到其他用户的愿望清单
17. 用户可以选择对购买的东西进行礼物包装
18. 用户可以选择附上礼物卡，并且可以在卡片上写上自己的信息
19. "报表查阅者"可以按照图书的类别、流量、最佳和最差销售情况来查看每天的购买报表
20. 用户必须经过适当的身份验证才能查看报表
21. 在网站上提交的订单最终必须与电话订单放入同一个订单数据库
22. 管理员可以在网站上增加新书
23. 在评论放上网站之前，必须经过管理员批准
24. 管理员可以删除书
25. 管理员可以编辑已有书的信息
26. 用户可以检查她最近的订单状态。如果订单还没有发货，她可以增加或移除书，更改送货方式、送货地址和信用卡信息
27. 系统能够支持500个并发用户

## 安装和运行

1. 安装依赖：
   ```
   pip install -r requirements.txt
   ```

2. 初始化数据库：
   ```
   python init_db.py
   ```

3. 运行应用：
   ```
   python app.py
   ```

4. 访问应用：
   打开浏览器访问 `http://localhost:5000`

## 技术栈

- Flask (Web框架)
- SQLite (数据库)
- SQLAlchemy (ORM)
- Flask-Login (用户认证)
- Flask-WTF (表单处理)
- Flask-Caching (缓存)

## 性能优化

应用已针对支持500个并发用户进行了优化：
- 使用数据库连接池
- 启用多线程
- 实现缓存机制
