## 🔧 文件功能说明

### 1. app.py - Web服务器
**功能**：
- 处理文件上传
- 调用生日转换逻辑
- 提供文件下载
- 管理临时文件

**主要路由**：
- `/` - 首页
- `/convert` - 处理转换请求
- `/cleanup` - 清理临时文件

### 2. birthday_converter.py - 转换引擎
**功能**：
- 农历转公历算法
- Excel文件处理
- 按月份拆分逻辑
- 列宽自动设置

**核心函数**：
- `process_excel_file_by_year()` - 主处理函数
- `convert_lunar_to_target_year_solar()` - 农历转公历
- `split_excel_by_month()` - 按月拆分

### 3. templates/index.html - 网页界面
**功能**：
- 文件上传表单
- 年份输入框
- 按月拆分选项
- 实时验证和提示

**特性**：
- 响应式设计
- 渐变色UI
- 加载动画
- 错误提示

### 4. 启动Web服务.bat - 一键启动
**功能**：
- 检查Python环境
- 安装依赖包
- 创建必要目录
- 启动Flask服务器

## 🚀 运行流程

```
用户操作                    系统响应
─────────────────────────────────────────────
1. 双击"启动Web服务.bat"  →  启动Flask服务器
2. 浏览器访问localhost:5000  →  显示网页界面
3. 选择Excel文件 + 输入年份  →  前端验证
4. 点击"开始转换"  →  上传文件到uploads/
5. 后台处理  →  调用birthday_converter.py
6. 生成结果文件  →  保存到downloads/
7. 自动下载  →  删除临时文件
```

## 🌐 网络配置

### 单机模式（默认）
```python
# app.py 最后一行
app.run(debug=True, host='127.0.0.1', port=5000)
```
只能本机访问：http://localhost:5000

### 局域网模式（当前配置）
```python
app.run(debug=True, host='0.0.0.0', port=5000)
```
局域网内所有电脑可访问：http://IP地址:5000

## 🛠️ 自定义配置

### 修改端口
编辑 `app.py` 最后一行：
```python
app.run(debug=True, host='0.0.0.0', port=8080)  # 改为8080
```

### 修改文件大小限制
编辑 `app.py`：
```python
app.config['MAX_CONTENT_LENGTH'] = 32 * 1024 * 1024  # 改为32MB
```

### 更换密钥
编辑 `app.py`：
```python
app.secret_key = 'your-new-secret-key'
```

## 🐛 故障排查

### 问题1：无法启动
**现象**：双击bat文件后闪退  
**解决**：
1. 检查Python是否安装：`python --version`
2. 手动运行查看错误：`python app.py`

### 问题2：无法访问网页
**现象**：浏览器打不开  
**解决**：
1. 确认服务已启动（查看命令行窗口）
2. 检查防火墙是否阻止5000端口
3. 尝试访问 http://127.0.0.1:5000

### 问题3：转换失败
**现象**：提示缺少必要列  
**解决**：
1. 检查Excel是否有"阳历生日"和"阴历生日"列
2. 列名必须完全匹配（区分大小写）

### 问题4：多人同时使用冲突
**现象**：文件名冲突或处理错误  
**解决**：
1. 每个用户上传的文件名会自动隔离
2. 如仍有问题，重启服务清理缓存
