# Flask应用部署指南

## 📋 部署方案总览

根据你的需求，提供以下几种部署方案：

1. **简单部署**（推荐小团队使用）- 直接复制运行
2. **生产环境部署**（推荐大团队使用）- 使用Waitress/Nginx
3. **Docker部署**（推荐专业运维）- 容器化部署

---

## 方案一：简单部署（最简单）⭐

适合：小团队（<20人）、内部使用、快速部署

### 步骤1：准备部署文件

在项目根目录创建 `deploy` 文件夹，包含以下文件：

```
deploy/
├── app.py                          # Flask主程序
├── birthday_converter.py           # 转换引擎
├── requirements_web.txt            # 依赖包列表
├── templates/
│   └── index.html                  # 网页界面
├── 启动Web服务.bat                 # Windows启动脚本
└── README_部署说明.md              # 本文件
```

### 步骤2：打包文件

**方法A：直接复制**
```bash
# 在源电脑上
# 复制以下文件和文件夹到U盘或网络共享文件夹
- app.py
- birthday_converter.py
- requirements_web.txt
- templates/ 文件夹
- 启动Web服务.bat
```

**方法B：压缩打包**
```bash
# 压缩成zip文件
压缩 "农历生日转换器Web版.zip" 包含：
- app.py
- birthday_converter.py
- requirements_web.txt
- templates/
- 启动Web服务.bat
```

### 步骤3：在目标电脑上安装

#### Windows系统

1. **解压文件**到任意目录（例如：`D:\农历生日转换器`）

2. **安装Python**（如果未安装）
   - 下载：https://www.python.org/downloads/
   - 建议版本：Python 3.8+
   - 安装时勾选 "Add Python to PATH"

3. **安装依赖包**
   ```bash
   # 打开命令行（cmd）
   cd D:\农历生日转换器
   
   # 安装依赖
   pip install -r requirements_web.txt
   ```

4. **启动服务**
   ```bash
   # 方法1：双击运行
   双击 "启动Web服务.bat"
   
   # 方法2：命令行运行
   python app.py
   ```

5. **访问应用**
   - 本机访问：http://localhost:5001
   - 查看IP地址：运行 `ipconfig`，找到IPv4地址
   - 其他电脑访问：http://目标电脑IP:5001

#### Linux系统

1. **上传文件**到服务器
   ```bash
   scp -r deploy/ user@server:/opt/birthday-converter/
   ```

2. **安装Python和pip**
   ```bash
   # Ubuntu/Debian
   sudo apt update
   sudo apt install python3 python3-pip
   
   # CentOS/RHEL
   sudo yum install python3 python3-pip
   ```

3. **安装依赖**
   ```bash
   cd /opt/birthday-converter
   pip3 install -r requirements_web.txt
   ```

4. **启动服务**
   ```bash
   # 前台运行（测试用）
   python3 app.py
   
   # 后台运行（生产用）
   nohup python3 app.py > app.log 2>&1 &
   ```

5. **配置防火墙**
   ```bash
   # 开放5001端口
   sudo ufw allow 5001/tcp    # Ubuntu
   sudo firewall-cmd --add-port=5001/tcp --permanent  # CentOS
   sudo firewall-cmd --reload
   ```

## 方案二：生产环境部署（推荐）⭐⭐⭐

适合：中大型团队、长期运行、需要稳定性

### 使用 Waitress（Windows/Linux通用）

Waitress是一个生产级别的WSGI服务器，比Flask开发服务器更稳定。

#### 步骤1：安装Waitress

```bash
pip install waitress
```

#### 步骤2：修改启动方式

创建 `run_production.py`：

```python
from waitress import serve
from app import app

if __name__ == '__main__':
    print("="*60)
    print("农历生日转换器 - 生产环境")
    print("="*60)
    print("访问地址: http://0.0.0.0:5001")
    print("="*60)
    
    serve(app, host='0.0.0.0', port=5001, threads=4)
```

#### 步骤3：启动服务

```bash
# Windows
python run_production.py

# Linux
python3 run_production.py
```

#### 步骤4：设置为系统服务（Linux）

创建 systemd 服务文件 `/etc/systemd/system/birthday-converter.service`：

```ini
[Unit]
Description=农历生日转换器
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/birthday-converter
ExecStart=/usr/bin/python3 /opt/birthday-converter/run_production.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启动服务：
```bash
sudo systemctl daemon-reload
sudo systemctl enable birthday-converter
sudo systemctl start birthday-converter
sudo systemctl status birthday-converter
```

## 方案三：Docker部署（最专业）⭐⭐⭐⭐

适合：专业运维、云部署、需要隔离环境

### 步骤1：创建Dockerfile

在项目根目录创建 `Dockerfile`：

```dockerfile
FROM python:3.9-slim

WORKDIR /app

# 复制依赖文件
COPY requirements_web.txt .

# 安装依赖
RUN pip install --no-cache-dir -r requirements_web.txt

# 复制应用代码
COPY . .

# 创建必要的目录
RUN mkdir -p uploads downloads

# 暴露端口
EXPOSE 5001

# 启动命令
CMD ["python", "app.py"]
```

### 步骤2：创建docker-compose.yml

```yaml
version: '3.8'

services:
  birthday-converter:
    build: .
    ports:
      - "5001:5001"
    volumes:
      - ./uploads:/app/uploads
      - ./downloads:/app/downloads
    restart: unless-stopped
    environment:
      - TZ=Asia/Shanghai
```

### 步骤3：构建和运行

```bash
# 构建镜像
docker-compose build

# 启动服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

## 🔧 常见问题解决

### 问题1：端口被占用

**现象**：`OSError: [Errno 98] Address already in use`

**解决**：
```python
# 修改 app.py 最后一行
app.run(debug=True, host='0.0.0.0', port=5002)  # 改为其他端口
```

### 问题2：防火墙阻止访问

**Windows**：
1. 控制面板 → Windows Defender 防火墙
2. 高级设置 → 入站规则 → 新建规则
3. 端口 → TCP → 5001 → 允许连接

**Linux**：
```bash
sudo ufw allow 5001/tcp
sudo ufw reload
```

### 问题3：依赖安装失败

**现象**：`pip install` 报错

**解决**：
```bash
# 升级pip
python -m pip install --upgrade pip

# 使用国内镜像源
pip install -r requirements_web.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题4：中文乱码

**解决**：确保文件编码为UTF-8
```bash
# Linux设置locale
export LANG=zh_CN.UTF-8
export LC_ALL=zh_CN.UTF-8
```

### 问题5：权限不足（Linux）

**解决**：
```bash
# 不要以root运行
sudo chown -R www-data:www-data /opt/birthday-converter
sudo chmod -R 755 /opt/birthday-converter
```

## 📊 性能优化建议

### 1. 增加并发处理能力

```python
# 使用Waitress替代Flask开发服务器
from waitress import serve
serve(app, host='0.0.0.0', port=5001, threads=8, connection_limit=100)
```

### 2. 添加Nginx反向代理

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:5001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 3. 定期清理临时文件

创建定时任务（Linux cron）：
```bash
# 每天凌晨2点清理
0 2 * * * find /opt/birthday-converter/uploads -type f -delete
0 2 * * * find /opt/birthday-converter/downloads -type f -mtime +1 -delete
```

## 🔒 安全建议

### 1. 修改密钥

```python
# app.py 中修改
import secrets
app.secret_key = secrets.token_hex(32)  # 生成随机密钥
```

### 2. 限制文件大小

```python
# app.py 中设置
app.config['MAX_CONTENT_LENGTH'] = 10 * 1024 * 1024  # 10MB
```

### 3. 添加访问日志

```python
import logging
from logging.handlers import RotatingFileHandler

handler = RotatingFileHandler('app.log', maxBytes=10000000, backupCount=10)
handler.setLevel(logging.INFO)
app.logger.addHandler(handler)
```

### 4. 使用HTTPS（生产环境必须）

使用Let's Encrypt免费证书：
```bash
sudo apt install certbot python3-certbot-nginx

```

## 🎯 快速开始（3分钟部署）

### Windows一键部署脚本

创建 `install_and_run.bat`：

```batch
@echo off
chcp 65001 >nul
echo 正在安装农历生日转换器...

REM 检查Python
python --version >nul 2>&1
if errorlevel 1 (
    echo 错误：未检测到Python，请先安装Python 3.8+
    pause
    exit /b 1
)

REM 安装依赖
echo 正在安装依赖包...
pip install -r requirements_web.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

REM 创建目录
mkdir uploads 2>nul
mkdir downloads 2>nul

REM 获取IP
for /f "tokens=2 delims=:" %%a in ('ipconfig ^| findstr /c:"IPv4"') do set IP=%%a
set IP=%IP:~1%

echo.
echo ========================================
echo 安装完成！
echo 本机访问: http://localhost:5001
echo 局域网访问: http://%IP%:5001
echo ========================================
echo.

REM 启动
python app.py
pause
```
