# 校务管理系统 - Python FastAPI版本
# 项目文件清单

## 核心文件

- main.py                 # FastAPI主应用入口
- config.py               # 配置文件（数据库、JWT密钥等）
- database.py             # 数据库连接和会话管理
- models.py               # SQLAlchemy数据模型定义
- schemas.py              # Pydantic数据验证模型
- auth.py                 # JWT认证和权限管理模块
- init_db.py              # 数据库初始化脚本

## API路由 (routers/)

- routers/student.py          # 学生管理API（增删改查、转学、离校）
- routers/teacher.py          # 教员管理API（增删改查、退休、调离）
- routers/score.py            # 成绩管理API（录入、统计、分析）
- routers/wage.py             # 工资管理API（5张工资表、配置、统计）
- routers/class_manage.py     # 班级管理API（增删改查、科目设置）
- routers/admin_user.py       # 管理员管理API（权限分配、用户管理）
