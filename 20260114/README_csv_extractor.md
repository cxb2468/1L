# CSV/Excel名称提取工具

这是一个基于PyQt5的图形界面工具，用于从CSV或Excel文件中提取名称列数据，并将其用逗号拼接显示。

## 功能特性

- 支持CSV和Excel文件格式
- 自动识别常见的名称列（如Name, 名称等）
- 如果没有标准名称列，可手动选择列
- 显示文件基本信息（路径、大小、行列数等）
- 预览文件内容
- 将提取的名称用逗号连接成一行输出
- **新增**: 一键复制提取结果到剪贴板（包含前缀文本）

## 使用方法

1. 启动程序：
   - 直接运行：`python csv_name_extractor.py`
   - 或双击运行批处理文件：`run_app.bat`

2. 在程序界面中：
   - 点击"选择CSV/Excel文件"按钮，选择要处理的文件
   - 程序会自动显示文件的基本信息和内容预览
   - 点击"提取名称列"按钮开始处理
   - 如果程序无法自动识别名称列，会弹出对话框让您手动选择
   - 提取的结果会显示在右侧的结果框中
   - 点击"复制结果"按钮将提取的内容（带前缀文本）复制到剪贴板

## 技术实现

- 使用pandas读取CSV和Excel文件
- 使用PyQt5构建图形界面
- 自动检测常见的名称列标题
- 提供手动选择列的功能
- 使用系统剪贴板实现复制功能
- 复制时自动添加提示文本

## 注意事项

- 支持的文件格式：CSV、XLS、XLSX
- 输出结果为逗号分隔的字符串格式
- 对于大型文件，加载和处理可能需要一些时间
- 复制功能会在状态栏显示确认消息
- 复制的内容包含提示文本："你是一个软件专家，请检查以下哪些软件 需要购买许可证才可以公司使用？"
