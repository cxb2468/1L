# 会议座次图生成器

这是一个基于 Flask 的 Web 应用，用于生成会议座位图表并安排参会人员座位。

## 功能特点

- 设置会场的行数、列数和过道位置
- 上传 Excel 文件导入参会人员名单
- 通过拖拽方式安排座位
- 实时显示座位分配记录
- 导出座位分配为 JSON 或 CSV 格式
- 导出座位图 PNG 图片或 PDF 文件

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python app.py
```

然后在浏览器中打开 `http://127.0.0.1:5000`

## 使用说明

1. 设置行数、列数和过道位置后点击"生成座次表"
2. 过道位置可以使用逗号分隔输入多个列号，例如："2,4"表示在第2列和第4列后添加过道
3. 上传Excel文件导入参会人员名单（第一列为姓名，第二列为单位）
4. 从左侧名单区域拖拽人员到座位上进行安排
5. 右侧区域实时显示座位分配记录
6. 可以随时调整布局并重新生成座次表
7. 使用下方按钮导出座次图为PNG图片或PDF文件

## Excel 文件格式

Excel 文件应包含两列：
- 第一列：姓名
- 第二列：单位

第一行会被忽略（作为表头）。
