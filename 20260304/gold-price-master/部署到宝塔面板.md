# 部署到宝塔面板指南

## 部署步骤

### 步骤1：登录宝塔面板
- 打开浏览器访问宝塔面板地址（通常为 `http://服务器IP:8888`  ）
- 使用账号密码登录

### 步骤2：创建网站
1. 进入网站管理 ：点击左侧菜单「网站」→「添加站点」
2. 填写站点信息 ：
   - 域名：填写你的域名（如无域名，可使用服务器IP）
   - 根目录：选择网站文件存放目录（如 www/wwwroot/gold-monitor ）
   - PHP版本：选择「静态」（因为是Python项目）
   - 其他选项保持默认，点击「提交」

### 步骤3：配置Python环境
1. 安装Python管理器 ：
   - 点击左侧菜单「软件商店」→ 搜索「Python项目管理器」→ 安装
2. 添加Python项目 ：
   - 点击左侧菜单「Python项目管理器」→「添加项目」
   - 项目路径：选择步骤2中创建的网站根目录
   - Python版本：选择 3.9+ （推荐3.9或3.10）
   - 框架：选择「Flask」
   - 启动方式：选择「Gunicorn」（生产环境推荐）
   - 点击「确认」

### 步骤4：上传项目代码
1. 进入文件管理 ：
   - 点击左侧菜单「文件」→ 进入网站根目录
2. 上传代码 ：
   - 方法1：使用「上传」功能，将本地项目文件（ main.py 、 config.py 、 templates/ 等）上传到根目录
   - 方法2：使用Git克隆（如果项目已托管在GitHub）：
     - 点击「终端」→ 执行命令：
       ```
       cd /www/wwwroot/gold-monitor
       git clone https://github.com/your-username/gold-price-monitor .
       ```

### 步骤5：安装项目依赖
1. 进入Python项目管理器 ：
   - 找到创建的项目，点击「管理」→「依赖管理」
2. 安装依赖 ：
   - 点击「批量安装」→ 输入 requirements.txt 中的依赖包：
     ```
     Flask
     DrissionPage
     requests
     PyInstaller
     ```
   - 或者直接执行命令：
     ```
     cd /www/wwwroot/gold-monitor
     /www/server/panel/pyenv/versions/3.9.10/bin/pip install -r requirements.txt
     ```

### 步骤6：配置启动服务
1. 修改启动配置 ：
   - 点击「Python项目管理器」→ 找到项目 → 点击「管理」→「配置」
   - 启动文件：填写 main.py
   - 应用名称：填写 app （对应 app = Flask(__name__) ）
   - 端口：填写 5000 （或其他未被占用的端口）
   - 点击「保存」
2. 启动服务 ：
   - 点击「启动」按钮，确保服务状态为「运行中」

### 步骤7：配置防火墙
1. 开放端口 ：
   - 点击左侧菜单「安全」→「防火墙」
   - 点击「添加规则」→ 端口填写 5000 → 备注「黄金监控」→ 点击「提交」

### 步骤8：测试访问
1. 访问配置页面 ：
   - 浏览器访问 `http://服务器IP:5000/config` 
   - 使用默认密码 admin888 登录
2. 访问价格页面 ：
   - 浏览器访问 `http://服务器IP:5000/` 
   - 查看黄金价格监控页面是否正常显示

### 步骤9：设置守护进程（可选）
为确保服务在服务器重启后自动启动：

1. 创建守护进程 ：
   - 点击左侧菜单「计划任务」→「添加任务」
   - 任务类型：选择「Shell脚本」
   - 执行周期：选择「开机执行」
   - 脚本内容：
     ```
     # 启动黄金监控服务
     cd /www/wwwroot/gold-monitor
     /www/server/panel/pyenv/versions/3.9.10/bin/gunicorn -w 4 -b 0.0.0.0:5000 main:app --daemon
     ```
   - 点击「保存」

### 注意事项
1. 依赖安装 ： DrissionPage 不需要安装浏览器驱动
2. 权限问题 ：确保项目目录权限正确，可执行 chmod -R 755 /www/wwwroot/gold-monitor
3. 日志查看 ：在「Python项目管理器」中点击「日志」可查看服务运行状态
4. 网络访问 ：确保服务器可访问互联网（用于获取黄金价格数据）

## 域名配置（可选）

### 步骤1：域名解析
1. 登录域名管理平台 ：
   - 进入你购买域名的服务商后台（如阿里云、腾讯云等）
   - 找到「域名解析」→「添加记录」
2. 添加解析记录 ：
   - 记录类型：选择 A 记录
   - 主机记录：填写子域名（如 gold 或 monitor ）
   - 记录值：填写服务器公网IP
   - TTL：保持默认，点击「确认」
3. 验证解析 ：
   - 等待1-10分钟后，使用 ping 命令验证域名是否解析成功：
     ```
     ping gold.your-domain.com
     ```

### 步骤2：在宝塔中添加域名
1. 进入网站管理 ：
   - 登录宝塔面板 → 点击左侧菜单「网站」
   - 找到之前创建的黄金监控站点，点击「设置」
2. 绑定域名 ：
   - 点击「域名管理」→「添加域名」
   - 输入你的域名（如 gold.your-domain.com ）
   - 点击「保存」

### 步骤3：配置反向代理
1. 进入反向代理设置 ：
   - 点击左侧菜单「网站」→ 找到站点 → 点击「设置」→「反向代理」
2. 添加反向代理 ：
   - 点击「添加反向代理」
   - 代理名称：填写「gold-monitor」
   - 目标URL：填写 http://127.0.0.1:5000 （Python服务运行的地址和端口）
   - 发送域名：选择你的域名
   - 其他选项保持默认，点击「提交」

### 步骤4：配置SSL证书（可选，推荐）
为了使用 https 访问（更安全）：

1. 申请SSL证书 ：
   - 点击左侧菜单「网站」→ 找到站点 → 点击「设置」→「SSL」
   - 选择「Let's Encrypt」→ 输入域名 → 点击「申请」
2. 强制HTTPS ：
   - 申请成功后，开启「强制HTTPS」选项

### 步骤5：测试域名访问
1. 访问配置页面 ：
   - 浏览器访问 `http://gold.your-domain.com/config`  （或 https:// ）
   - 输入默认密码 admin888 登录
2. 访问价格页面 ：
   - 浏览器访问 `http://gold.your-domain.com/` 
   - 确认黄金价格监控页面正常显示

### 步骤6：优化配置（可选）
1. 修改Python服务端口 ：
   - 如果需要修改Python服务端口（默认5000），可在「Python项目管理器」中修改
   - 修改后需要同步更新反向代理的「目标URL」
2. 配置Gunicorn参数 ：
   - 在「Python项目管理器」中，点击「管理」→「配置文件」
   - 优化Gunicorn参数（如工作进程数、超时时间等）：
     ```
     workers = 4
     timeout = 30
     bind = 0.0.0.0:5000
     ```

## 常见问题排查

### 1. 域名无法访问 ：
   - 检查域名解析是否生效（可使用 nslookup 命令验证）
   - 检查宝塔防火墙是否开放80/443端口
   - 检查反向代理配置是否正确

### 2. 服务启动失败 ：
   - 检查Python依赖是否安装完整
   - 检查项目代码是否有语法错误
   - 查看「Python项目管理器」中的日志信息

### 3. 访问速度慢 ：
   - 优化Gunicorn工作进程数（根据服务器CPU核心数调整）
   - 启用宝塔的「网站加速」功能

## 总结

通过以上配置，你可以使用域名（如 gold.your-domain.com ）访问黄金价格监控系统，无需再使用IP地址和端口号。配置页面和价格页面均可通过域名正常访问，系统将在后台持续监控黄金价格并发送预警消息。

访问地址汇总 ：

- 配置页面： `https://gold.your-domain.com/config` 
- 价格页面： `https://gold.your-domain.com/` 
- 管理页面： `https://gold.your-domain.com/config` 

（将 gold.your-domain.com 替换为你实际的域名）
