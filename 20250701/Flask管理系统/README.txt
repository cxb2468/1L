Flask管理系统使用说明
=======================

系统要求：
- Windows 7/8/10/11 或 Linux/macOS
- 无需安装Python环境（已包含在打包文件中）

功能特性：
- 合同管理：支持合同信息录入、编辑、删除、附件上传
- 项目管理：支持项目信息管理、多文件附件上传
- 人员管理：支持员工信息管理、附件上传
- 收付款管理：支持收款、付款记录管理、附件上传
- 动态字段：支持自定义字段配置
- 数据导入导出：支持Excel格式数据导入导出
- 权限管理：支持用户角色和权限控制

首次使用：
1. 双击"初始化系统.bat"（Windows）或运行"./init_system.sh"（Linux/Mac）
2. 等待初始化完成
3. 双击"启动系统.bat"（Windows）或运行"./start.sh"（Linux/Mac）

使用方法：

Windows用户：
1. 双击"启动系统.bat"
2. 在浏览器中访问: http://localhost:5000
3. 使用默认账户登录：
   用户名: admin
   密码: admin123
4. 按 Ctrl+C 停止服务

Linux/Mac用户：
1. 打开终端，进入此目录
2. 首次使用：运行 ./init_system.sh
3. 启动系统：运行 ./start.sh
4. 在浏览器中访问: http://localhost:5000
5. 按 Ctrl+C 停止服务

附件功能说明：
- 合同附件：按合同名称创建文件夹存放，支持单个文件上传
- 项目附件：支持多文件上传，按项目名称创建文件夹存放
- 人员附件：支持员工相关文件上传
- 收付款附件：按合同名称创建文件夹存放

附件管理功能：
- 上传：支持多种文件格式（PDF、Word、Excel、图片等）
- 下载：点击文件名可下载附件
- 删除：可单独删除不需要的附件
- 预览：支持图片文件在线预览

修改端口：
1. 用文本编辑器打开 config.py 文件
2. 找到 PORT = 5000 这一行
3. 将 5000 改为您想要的端口号
4. 保存文件并重新启动应用

注意事项：
- 首次运行必须先运行初始化脚本
- 首次运行可能需要防火墙授权
- 确保data.db文件存在且可读写
- 如需备份数据，请复制data.db文件
- 请及时修改默认管理员密码
- 附件文件存储在uploads目录下，请定期备份
- 项目附件支持多文件上传，建议单个文件不超过10MB

常见问题：
1. 中文显示乱码：启动脚本已设置UTF-8编码
2. 连接被拒绝：应用已配置为监听所有网络接口
3. 端口被占用：请修改config.py中的端口号
4. 登录失败：请先运行初始化脚本
5. 附件上传失败：检查uploads目录权限和磁盘空间
6. 项目编辑页面卡住：检查是否有大文件上传或网络问题
